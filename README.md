## A simple rule-validation API.

## Usage

### Clone

```
 git clone https://github.com/Segun98/rule-validation-flw.git
```

### Install

```
 npm install
```

### Start

```
npm run start
```

### Run on http://localhost:8080

### API Routes

```
GET: /

POST: /validate-rule
```
